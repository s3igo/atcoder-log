# AtCoder Rust

Log of My AtCoder answers written in Rust.

## Requirements

- direnv
- rustup

## Usage

```sh
# Install cargo-compete
make install

# Init project
make init

# Update cargo-compete
make update

# Logout from AtCoder
make logout

# Check format with nightly rustfmt feature
make check

# Create new contest directory
make new ARG=<contest-id>
```
